Comandos comunes para git y la terminal

Comandos de git

Comando Descripción
git status Muestra el estado actual del repositorio
git clone {ssh url} Crea una copia de un repositorio de manera local
git add {archivo or carpeta} Rastrea el archivo y lo añade a la staging area
git add * Hace el add con todos los archivos
git commit -m “{Descripción del commit}” Registra los cambios en el repositorio local
git push origin {branch} Manda los cambios del repositorio local al remoto
git restore --staged {archivo} Quita el archivo de la staging area
git restore {archivo} Descarta cambios de un archivo
git checkout -b {nombre} Crea una nueva branch y te mueve automaticamente a esta
git branch {nombre} Crea una nueva branch, sin embargo no te mueve a esta
git checkout {branch a la que se quiere ir}
git switch {branch a la que se quiere ir} 		Permiten moverse entre branches
git branch Muestra todas las branches del repositorio
* Señala la branch actual
git pull origin main Para asegurarnos de tener todos los cambios recientes en la main branch
git merge {nombre del branch} Hace merge de los contenidos del branch a con la main branch
git push Crea un commit y lo desplaza al repositorio remoto
git branch -d {nombre del branch} Borra la branch indicada de manera local
git branch -D {nombre del branch} Borra la branch sin importar si la branch no esta fusionada, solo de manera local
git push origin -d {nombre del branch} Borra la branch del repositorio remoto

Comandos de terminal

Comando Descripción
ls Mostrar los archivos del directorio donde esta la terminal
ls -a Muestra todos los archivos, hasta los ocultos
ls -al Muestra todos los archivos en forma de lista
cd Cambiar de direccion dentro del directorio
cd.. Pemite regresar un directorio atras
mkdir {nombre} Crear una nueva carpeta
touch {nombre} Crear un archivo
vim {archivo} Editar un archivo desde la terminal
i Ingresa al modo insert lo que permite editar el contenido
:w Guarda el contenido
:q Sale de editor
:wq Guarda y sale del editor
cat {archivo} Ver el contenido de un archivo
man {comando} Ver el manual de un comando
history Muestra el historial
code . Abre Visual Studio Code en la dirección de la terminal
cp {archivo a copiar} {donde se copia} Copiar un archivo a otro archivo o carpeta
rm {archivo} Borra un archivo
mv {archivo} {a donde mover} Mueve un archivo a otra carpeta
Usar *.{tipo} permite mover todos los archivos de cierto tipo
rm -r {carpeta} Borra una carpeta
head/tail {archivo} -n Permite ver las primeras o ultimas lineas de un archivo respectivamente
less Para ver el archivo por secciones
grep -i Busca una palabra dentro del archivo
